Leer ficheros Markdown con colores y subrayado en la terminal

21 November 2015 on Markdown

Si bien los ficheros Markdown (.md) se pueden leer con cualquier visor de texto, la mayoría no los muestran como texto con formato sino como texto plano.

mad es un visor de Markdown fácil de instalar y de utilizar, capaz de formatearlos con colores y subrayado.

Instalación

git clone https://github.com/tj/mad.git
cd mad
sudo make
sudo make install

Configuración

Edita el fichero /usr/local/etc/mad.conf con vim o con cualquier otro editor de texto:

sudo vim /usr/local/etc/mad.conf

Debe quedar así:

heading: 1;4;36m
code: 1;33m
strong: 1;4;35m
em: 1;35m

Guarda los cambios.

Utilización

mad <fichero .md>
